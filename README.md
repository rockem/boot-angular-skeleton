# Spring boot + AngularJS skeleton

This is a simple skeleton for a web app based on spring boot

## Technologies
- Gradle
- Spring boot
- AngularJS
- MongoDB
- Selenium
- Cucumber JVM (Groovy)
- Less

## How to use it?
#### Run the backend
./gradlew bootRun
#### Start mongo for the backend
./gradlew startMongo
#### Stop backend's mongo
./gradlew stopMongo
#### Run the acceptance tests
./gradlew runFeatures
